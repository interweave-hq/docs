# Fields

`fields` is an object that contains the outline of your schema. The properties will be the fields within your data object.

```js {3, 8}
const config = {
	key: ...,
	fields: {
		"firstName": {
			schema: {
				type: "string";
			}
		},
		"lastName": {
			schema: {
				type: "string"
			}
		}
	}
}
```
