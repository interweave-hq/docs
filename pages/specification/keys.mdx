# Keys

`keys` is an object that contains the outline of your schema. The properties within will be the keys of each field within your data object.

```js {3, 8}
const config = {
	keys: {
		"firstName": {
			schema: {
				type: "string";
			}
		},
		"lastName": {
			schema: {
				type: "string"
			}
		}
	}
}
```
