# Do It Yourself

Interweave doesn't require the npm package to function. You can make a regular HTTP request to the Interweave API to build your interface.

Here is a basic script to do that:

```js
// Push to API
try {
  const res = await fetch(
    `https://api.interwv.com/api/v1/projects/__YOUR_PROJECT_ID__/interfaces`,
    {
      method: "POST",
      body: JSON.stringify({
        // Schema information
        schema_config: schema,
        // Note that key/slug, title, and description are separate here
        key: schema.slug,
        title: schema.title,
        description: schema.description
      }),
      headers: {
        "Content-Type": "application/json",
        authorization: `Bearer __YOUR_PROJECT_API_TOKEN__`
      }
    }
  );
  const data = await res.json();
  console.log(`Update responded with a status of ${data.http_status}.`);
} catch (err) {
  console.log(err);
}
```
